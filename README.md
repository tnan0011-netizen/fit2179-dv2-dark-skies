# Where Did the Stars Go? Mapping Australia's Darkest Skies

FIT2179 Data Visualisation 2 starter project.

## Important
The current CSV files are placeholder data only so the page loads and the Vega-Lite structure works.
Replace them with real data before submission.

## Folder structure
- `index.html` — main webpage
- `css/styles.css` — page styling
- `js/vega_lite_vis.js` — embeds Vega-Lite specs
- `visualisations/` — separate Vega-Lite JSON files
- `data/` — CSV data files
- `sketch/` — upload paper sketch PDF here
